# HuffmanArchiver

Консольная утилита для сжатия и распаковки файлов, использующая алгоритм Хаффмана. Проект реализован на языке C.

## Описание
Программа выполняет архивацию файлов на основе частотного анализа символов и построения оптимального префиксного кода (дерево Хаффмана). Это позволяет эффективно сжимать данные без потерь.

## Основные возможности
* **Сжатие файлов**: уменьшение размера файла с помощью алгоритма Хаффмана.
* **Распаковка**: восстановление оригинального содержимого из сжатого файла.
* **Побитовая работа**: для максимальной эффективности упаковки данных используются специальные модули `binary_reader` и `binary_writer`.

## Структура проекта
Проект логически разделен на модули:
* **`main.c`** — точка входа, обработка пользовательского ввода.
* **`parser.c/h`** — разбор аргументов командной строки.
* **`huffman_alg.c/h`** — основная логика алгоритма сжатия/распаковки.
* **`huffman_builder.c/h`** — построение дерева Хаффмана.
* **`huffman_tree.c/h`** — работа с узлами и обход дерева.
* **`binary_reader.c/h`** и **`binary_writer.c/h`** — побитовое чтение и запись файлов.

## Сборка и использование

### Сборка
Выполните команды в корневой папке проекта:
```bash
> make 

# Пример сжатия
> ./huffman_archiver -c input.txt output.huff

# Пример распаковки
> ./huffman_archiver -d output.huff restored.txt
```
